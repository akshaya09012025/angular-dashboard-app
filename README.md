# Angular Dashboard App

This Angular 14 project implements a **User Dashboard** with a table, a Chart.js pie chart, and a lazy-loaded user form.

## Features

- **User Table:** Displays Name, Email, and Role of users. Shows placeholder when no users exist.
- **Chart.js Pie Chart:** Shows role distribution (Admin, Editor, Viewer). Updates dynamically.
- **Lazy-loaded User Form:** Modal for adding users to dashboard.
- **RxJS State Management:** Users managed via BehaviorSubject for reactive updates.
- **Form Validation & Error Handling:** Required fields, email validation, and modal cancel handling.
- **Design Theme:** Colors #1c4980 and #383838. Button/input height 48px.
- **Responsive CSS:** Proper alignment for form fields, table, and chart.
